# Capstone

This repository currently contains a ready-to-run backend and a `frontend/` package that now ships a TypeScript SDK for the implemented backend API.

## Backend

Use the backend guide for full local setup and daily development:

- [Backend setup guide](./backend/README.md)

The backend guide includes:

- starting the MariaDB database with Docker
- configuring the backend environment
- running migrations
- seeding initial data
- starting the CodeIgniter server
- default seeded accounts
